MongoDB

1. How check MongoDB version?
	command - mongod --version

2. How check MongoDB server is running or not? and also use to enter MongoDB shell to use MongoDB command like, show databases.
	command - mongosh

3. How to start MongoDB server?
	command - net start MongoDB

4. How to stop MongoDB server?
	command - 1. ".exit" -> terminate the current CMD.EXE process or an active batch script.
		  2. net stop MongoDB
